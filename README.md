# Employee-Request-Management-System

A production-ready SaaS-based Flow Management System designed to automate employee request workflows between Employees, POCs, and Managers.  

The system streamlines request tracking, approvals, reporting, and administrative control through a scalable architecture.

---

## 🚀 Overview

Flow Management System is a multi-role SaaS application that enables organizations to:

- Submit and manage employee requests
- Automate approval workflows
- Generate Excel-based reports
- Monitor request lifecycle status
- Maintain role-based access control
- Track performance and analytics

The system is built with a scalable backend, secure authentication, and a modern frontend.

---

### Additional Features
- Excel Report Generation
- Role-Based Access Control (RBAC)
- Image/File Handling
- API Validation & Middleware
- Production Error Handling

---

## 👥 User Roles

### 1. Employee
- Submit requests
- Track request status
- View request history

### 2. POC (Point of Contact)
- Review assigned requests
- Approve / Reject requests
- Forward to Manager

### 3. Manager
- Final approval
- View reports
- Export Excel summaries

### 4. Admin
- Manage users
- Assign roles
- Monitor system activity
